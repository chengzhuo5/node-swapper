# Node Swapper

这是一款在 windows 下改变 Node 版本的命令行工具，基于环境变量实现。

## 安装

在管理员权限下运行：

`npm i node-swapper -g`

## 安装 Node 版本

在管理员权限下运行：

```javascript
node-swapper install 15.10.0
```

目前版本号需要精确匹配，未来会支持模糊匹配。

## 卸载 Node Swapper

在管理员权限下运行：

```javascript
node-swapper uninstall
npm uninstall node-swapper -g
```
